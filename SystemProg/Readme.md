# Самостійна робота

## Програма по архівації файлів
- Можливість односасної архівації декількох папок.
- Має бути перевірка на вкладеність папок (перехресна).
- виводи логи архівації в окремих вікнах.
- записувати загальний лог архівації в  файл.
- в якості архіватора використати tar.

```C#
         // tarfileName - назва архіва, наприклад temp.tar.gz
         // folderName  - папка яку архівуємо.
         if (!Run($"tar -czvf {tarFileName} {folderPath}", tempFolder))
                throw new Exception("Error pack !");
  
```
